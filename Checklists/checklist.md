# Test Checklist

## 1. Login / User Authentication
- Open login page
- Log in with valid username and password
- Log in with invalid username or password
- Verify error message is displayed for invalid credentials
- Log out from the application

## 2. Product List / Catalog
- Verify products are displayed on the products page
- Verify product image, name, and price are shown
- Open product details page and verify product information

## 3. Cart / Checkout
- Add product to the cart
- Verify product appears in the cart
- Verify cart updates correctly after removing a product
- Proceed to checkout
- Verify checkout page opens successfully
- Fill in checkout form with valid data
- Verify order confirmation is displayed

## 4. UI / UX
- Verify website loads within acceptable time
- Verify navigation is clear and easy to use
- Verify buttons and links work correctly
- Verify error messages are readable and clear

## 5. Non-Functional / Security
- Verify website is responsive on different screen sizes
- Verify website uses HTTPS connection
- Verify website works correctly in Chrome, Firefox, Edge, and Safari
