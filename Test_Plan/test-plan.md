# Test Plan

## 1. Scope

**In Scope:**
- User login functionality
- Viewing list of products
- Sorting products by price, name, or rating
- Adding products to cart and proceeding to checkout

**Out of Scope:**
- Admin panel functionality
- Payment through external gateways (if not available in demo)
- Integration with other systems


## 2. Approach

**Testing Approach:**
- Manual testing will be used to verify all functionality
- Types of testing:
  - **Functional Testing:** Verify login, product listing, sorting, cart, and checkout functionality
  - **UI/UX Testing:** Verify correct display of elements, buttons, and user-friendly error messages
  - **Non-Functional Testing:** Check website loading speed, mobile responsiveness, and HTTPS security

**Test Design:**
- Test cases will be created for each feature
- Positive and negative scenarios will be included
